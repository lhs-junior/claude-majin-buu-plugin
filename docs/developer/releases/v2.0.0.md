## Upgrade Guide

### From v1.x to v2.0.0

1. **Update package**:
   ```bash
   npm update -g flux
   # or
   git pull origin main
   npm install
   npm run build
   ```

2. **Update documentation links**:
   - Check your bookmarks and internal references
   - Use new 3-tier structure: `docs/{user,developer,ai}/`

3. **Enable Fusion features** (opt-in):
   ```typescript
   // Fusion is enabled by default, but you can configure:
   import { HooksManager } from 'flux/fusion';

   const hooks = new HooksManager();
   hooks.registerHook('PostToolUse', async (context) => {
     // Custom hook logic
   });
   ```

4. **Review absorbed features**:
   - Findings: Track decisions and discoveries
   - Progress: Auto-log session activities
   - Memory: Auto-categorization enabled
   - Worktree: Use for isolated branch work

5. **Test your integration**:
   ```bash
   npm test
   ```
